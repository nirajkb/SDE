# Documentation

This folder contains all project documentation for the Ad Click Charging System.

   - Component diagrams

   - Design decisions

2. **detailed-design.md**

### Project Report (`report/`)

1. **technical-report.md**
   - Complete project report
   - Problem statement and solution
   - Implementation details
   - Testing results
